static method addConnection  [line 40]

  static void addConnection( array $connection  )

Adds the new connection information to the list of available connections.

$connection is an array containing the elements: string name = Name of the connection (eg. 'default') string driver = PDO driver to be used (eg. 'mysql') string dsn = Connection string string username = Username string password = Password string host = Where the database server is located (eg. 'localhost') string database = Name of the actual database

If you use 'dsn', then you don't need 'host' and 'database', and vice-versa.

 Parameters:
array   $connection:  Database connection information
