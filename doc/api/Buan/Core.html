	static method configure  [line 24]





	

    

    
 
		static void configure(
string
$configPath 
)
    

    



	
This method prepares the Buan environment by reading and acting on a set of  configuration scripts found in $configPath.



$configPath should contain the following script:  app.php		= Set config variables, db connections and Model relations  bootstrap.php	= An optional script that performs your custom routines
The following tasks are executed in order: 
  1. "app.php" is executed.
  2. Any extensions defined in "app.extensions" are initialised.
  3. "bootstrap.php" is executed.

			Parameters:

			

					

				string  
				
				$configPath: 
				
				 Absolute path to folder containing your configuration files
