Class ModelCollection

Class Overview
Class Overview | Property Summary | Properties Detail | Method Summary | Methods Detail

Implements interfaces:

  • Iterator (internal interface)
  • ArrayAccess (internal interface)
  • Countable (internal interface)

An array substitute which provides a few additional methods for performing some db actions on a group of Models.

NOTE: Won't be able to perform certain functions such as implode() on instances.
