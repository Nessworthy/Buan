Class Html

Class Overview
Class Overview | Property Summary | Properties Detail | Method Summary | Methods Detail

Provides a number of methods specific to HTML Views.

Example usage (from with your custom View class' constructor):

  1.     $this->html = new Buan\ViewHelper\Html($this);

Example usage to add CSS/JS sources to your template:

  1.     $this->html->addStylesheets('/path/to/file.css', '/path/to/another.css');

Example usage to render the CSS/JS <link>/<script> tags in your template:

  1.     echo $this->html->getStylesheetsView()->renderAfter($this);

If you want all URLS generated by ::eUrl() to be suffixed with a string  (eg. ".html"), then set that suffix here.



Note that the suffix will be visible in the [app.command.requested] config  variable so it's your application's responsibility to strip this suffix from  the URL and re-store it in [app.command.requested].

Adds external stylesheet sources to the View.



The first argument given is special and can take the form of either a URL or  one of the following key phrases: screen | print | all | screen,print
If one of these phrases is used then it will be included in the "media"  attribute of the <link> nodes.
  1.  $html->addStylesheets('/css/resource1.css', '/css/resource2.css');
    2. 

  2.  $html->addStylesheets('print', '/css/resource3.css');
    3.
