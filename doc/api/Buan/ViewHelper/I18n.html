Class I18n

Class Overview
Class Overview | Property Summary | Properties Detail | Method Summary | Methods Detail | Constants Summary | Constants Detail

Enables specified strings within a View's buffer to be translated to any other language.

Example usage (from within your custom View class' constructor):

	translateView  [line 125]





	

    

    
 
		void translateView(
)
    

    



	
Translates all [t:...] strings in $this->view's buffer to the language  specified by $this->language. The dictionary for this language will be  searched for in the list of $this->dictionaryFolders.



Examples of strings and their conversions:  
  1.  [t:Hello] ... Hello
    2. 

  2.  [t:Dave:Hello, %s] ... Hello, Dave
    3. 

  3.  [t:Foo:Bar:I'm %s, and you're %s] ... I'm Foo, and you're Bar
    4. 

  4.  [t:See\: www.thundermonkey.net] ... See: www.thundermonkey.net
    5.
